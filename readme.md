## Descripción
Juego Piedra, Papel o Tijera en JavaScript con manejo de estados y eventos.
En este pequeño proyecto se utiliza localStorage para guardar el estado del juego.

## Tecnoogías
- HTML
- CSS
- TypeScript

## Aprendizaje
- Manejo de estados
- Manejo de eventos
- LocalStorage
- TypeScript
